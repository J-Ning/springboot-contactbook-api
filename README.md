# Sample - Spring Boot Contact Book Application

A Spring Boot application demonstrating:
- **JPA** (Spring Data JPA with PostgreSQL)
- **MyBatis** mapper-based data access
- **REST API** with full CRUD operations
- **Threading concepts**: deadlocks, synchronized methods, ReentrantLock
- **Transactional management**: rollback demonstrations
- **Matrix multiplication** and **selection sort** algorithms
- **Network segment** calculation

## Tech Stack
- Java 21, Spring Boot 3.3.2
- PostgreSQL, Spring Data JPA, MyBatis
- Lombok

## Running
```bash
mvn spring-boot:run
```
Server runs on port **8082**.
